# mylist

Mylist of YouTube, NicoVideo

```bash
./mls import https://www.youtube.com/watch?v=zSXRMrtyNI4
./mls play
```

## Requirements

- Bash
- Ruby
- peco

## Mylist

Mylist is stored as a `tsv` file (default `~/.mls` or set `$MLS_PATH`).

## Demo

`sample.playlist` is a sample.

```bash
MLS_PATH=sample.playlist ./mls play
```
